# Flask Docker Practice

This is a mini-beginner-friendly Flask project containerised with Docker.

## Steps Completed

1. **Created a simple Flask app** in `src/app.py`.
2. **Dockerized the app** with Dockerfile.
3. **Built and ran the Docker image** to test locally.
4. **Created docker-compose.yml** to orchestrate containers.
5. **Pushed to GitHub**.

## Run Project

```bash
docker compose up -d
